contrainer เป็นการกำหนดพื้นที่
  • contrainer = พื้นที่จะอยู่ตรงกลางหน้าจอ
  • contrainer-fluid = พื้นที่จะเต็มหน้าจอ
  • contrainer-{breakpoint}
หัวข้อที่ 1 - จัดการข้อความและตัวอักษา
  • Heading (H1-H6) กำหนด class = 'h1-h6'
  • display (display 1-6) กำหนด class = 'display-(1-6)'
  • lead ทำตัวหนา และ เว้นระยะด้านบนและล่าง class = 'lead'
  • กำหนดความนาตัวอักษา class = ""
    • fw-bold
    • fw-bolder
    • fw-normal
    • fw-light
    • fw-lighter
    • fst-italic
  • กำหนดขนาดและรูปแบบข้อความ class = ""
    • text-uppercase = พิมพ์ใหญ่
    • text-lowercase = พิมพ์เล็ก
    • text-capitalize = ตัวแรกพิมพ์ใหญ่
    • fs-(1-6) = ขนาดข้อความ
  • Blockquote ทำบล๊อคส่วนหนึ่งของข้อความ class = 'blockquote'
  • List
    • list-unstyleed = ไม่กำหนดรูปแบบ
    • list-inline = เรียงในแนวนอน
      • แล้วใส่ class = 'list-inline-item' ใน li ทุกตัว

หัวข้อที่ 3 - Alignment & Display
Text alignment ใช้ class
  • text - start
  • text - center
  • text - end
display ใส่ class
  • d-none - ไม่มีการแสดงผล
  • d-block - เรียงแนวตั้ง
  • d-inline - แนวนอน
  • d-inline - block - แนวนอนและขยายพื้นที่ด้านใน
  • d-block - แปลงเป็น block
หัวข้อที่ 4 - Float & Flixed position
  • Float- [start , end , none] เพิ่ม class clearfix จะทำให้เนื้อหาอื่นไม่ขึ้นมาแทรก
  • Fixed-[top , bottom , sticky-top (เนื้อหาจะติดขอบเมื่อเลื่อนลงมาถึง)]
  • visible / invisible
หัวข้อที่ 5 - Margin
กำหนดระยะห่างภายนอก element กับวัตถุอื่น ใช้ class = m[position]-[size]
Ex. mb-0 , mt-1 , ms-2 , me-3
Position
  • t = top
  • b = bottom
  • s = left
  • e = right
  • x = left , right
  • y = top , bottom
  • blank = 4 ทิศทาง
size
  • 0 = (x=0 , y=0)
  • 1 = (x= spancer * 0.25 , y= spancer * 0.25)
  • 2 = (x= spancer * 0.5 , y= spancer * 0.5)
  • 3 = (x= spancer , y= spancer )
  • 4 = (x= spancer * 1.5 , y= spancer * 1.5)
  • 5 = (x= spancer * 3 , y= spancer * 3)
  • Auto
หัวข้อที่ 6 - Padding
กำหนดระยะห่างภายใน element ใช้ class = p[position]-[size]
Ex. pb-0 , pt-1 , ps-2 , pe-3
หัวข้อที่ 7 - กำหนดขนาด
  • w (width) กำหนดความกว้าง
  • h (height) กำหนดความสูง
