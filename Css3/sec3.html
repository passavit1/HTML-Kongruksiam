กำหนดความกว้างและความสูง

width ใช้กำหนดความกว้าง
  • auto คือ broweser กำหนดเอง
  • length คือ ความกว้างแบบหน่วยวัด PX CM เป็นต้น
  • % คือ ความกว้างแบบ %
height ใช้กำหนดความสูงของวัตถุที่ต้องการ
  • auto คือ broweser กำหนดเอง
  • length คือ ความสูงแบบหน่วยวัด PX CM เป็นต้น
  • % คือ ความสูงแบบ %
ความกว้างความสูงแบบ Max min
  • width-min ใช้กำหนดความกว้างต่ำสุด
  • width-max ใช้กำหนดความกว้างสูงสุด
  • height-min ใช้กำหนดความสูงต่ำสุด
  • height-max ใช้กำหนดความสูงสูงสุด
หน่วยที่รองรับ
  • none ไม่กำหนดค่า
  • length กำหนดแบบหน่วยวัด
  • % กำหนดเป็นน %

กรณีภาพซ้ำ

ใช้ background-repeat : xxx ; โดยจะแบ่งเป็น 4 หัวข้อย่อยดังนี้
  • repeat : (ค่า default : ทำให้ภาพเต็มพื้นที่ของ Element
  • repeat-x : จัดเรียงเฉพาะแนวแกน x
  • repeat-y : จัดเรียงเฉพาะแนวแกน y
  • no-repeat : แสดงภาพพื้นหลังภาพเดียว
