Media Query
คือ การเขียนรูปแบบ Style ให้แสดงผลตามหน้าจอที่แตกต่างกัน

ความกว้างอุปกรณ์
  • 320px - 480px : Mobile devices
  • 481px - 768px : Ipads , Tablets
  • 769px - 1024px : laptops
  • 1025px - 1200px : Desktops
  • 1201px + : TV , Widescreen

Viewport Units
องค์ประกอบ
  • VW = ความกว้าง
  • VH = ความสูง
  • vmin = ค่า % ต่ำสุดของ viewport
  • vmax = ค่า % สูงสุดของ viewport
