กำหนดเงาด้วย box - shadow

box - shadow : x y blur spread color
  • X : เงาแกน x (+ ขวา , - ซ้าย)
  • Y : เงาแกน y (+ ล่าง , - บน)
  • Blur : ขนาดความมัวของเงา
  • Spread : ขนาดของเงา
  • Color : สีของเงา
Example


box-shadow: 10px 10px 20px 3px red;

box-shadow: 5px 5px 30px 2px blue, -5px -5px 30px 2px orange;
