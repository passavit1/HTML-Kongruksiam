Flexbox


ช่วยจัดวางตำแหน่ง element ต่างๆในหน้าเว็บ โดยมีคุณสมบัติดังนี้
  • จัดเรียงตำแหน่งของ Element ได้ง่ายขึ้น
  • กำหนดขนาดให้พอดีกับพื้นที่ว่างแบบอัตโนมัติ

องค์ประกอบของ Flexbox
  • container = กรอบที่ครอบ Item
  • Item = สมาชิกของ container
ค่าเริ่มต้น flex จะเป็นแนวนอน

กำหนดทิศทางด้วน Flex - direction
  • Row = แนวนอน ทิศเดียวกับแกนหลัก
  • Colume = แนวตั้ง ทิศเดียวกับแกนหลัก
  • Row-reverse = แนวนอน ทิศตรงข้ามกับแกนหลัก
  • Colume-reverse = แนวนอน ทิศตรงข้ามกับแกนหลัก

กำหนดขนาดด้วย Flex-wrap
กรณีที่ขนาด Item ใหญ่กว่าพื้นที่ container
  • nowrap = จัดวาง Item ที่เกินพื้นที่ไปด้านขวามือ
  • wrap = จัดวาง Item ที่เกินพื้นที่เรียงจาก บน ลง ล่าง
  • wrap-reverse = จัดวาง Item ที่เกินพื้นที่เรียงจาก ล่าง ขึ้น บน

Flex - properties (items)
  • flex - shrink = ให้ item หดตัวเท่าใดเมื่อเทียบกับ item อื่น ค่าเริ่มต้น = 1
  • flex - grow = ให้ item ขยายตัวเท่าใดเมื่อเทียบกับ item อื่น ค่าเริ่มต้น = 0
  • flex - basis = กำหนดค่าความยืดหยุ่นเบื้องต้น
  • flex:1 = ทำให้ item ในแถวเดียวกันมีขนาดเท่ากัน
Justify-content
เทียบกับแกนหลัก
  • flex-start = ชิดซ้ายทิศทางตามแนวนอน
  • center = ตรงกลาง
  • flex-end = ชิดขวาทิศทางตามแนวนอน
  • space-between = ระยะห่างซ้าย ขวา และขนาด item เท่ากัน (ติดมุม)
  • space-around = ระยะห่างซ้าย ขวา และขนาด item เท่ากัน
Flex Alignment
  • stretch = ขนาด item เท่ากับ container
  • Flex-start = ด้านบน container
  • center = กลาง container
  • flex-end = ด้านล่าง container
  • align-self = ปรับแต่งเฉพาะบาง item
