CSS 3

Introduction
Css (Cascading style sheets) เป็นเทคโนโลยีใช้จัดรูปแบบและการแสดงผลของเว็บ

โครงสร้าง

มีส่วนประกอบอยู่สองส่วน Selector และ Declaration
  • Selector คือ Tag HTML ID Class
  • Declaration ใช้กำหนดค่า Selector

Selector

คือ Tag HTML ID CLASS ที่ต้องการกำหนดคุณสมบัติ
  • Element Selector คือ การกำนหด Style ให้ TAG HTML โดยตรง
  • Class Selector คือ การกำนหด Style ให้ Class โดยใช้สัญลักษณ์ (.)
  • ID Selector คือ การกำนหด Style ให้ ID โดยใช้สัญลักษณ์ (#)

Declaration

ใช้กำหนดค่าให้กับ Selector มีส่วนประกอบดังนี้
  • Property หรือ Attribute คือ คุณสมบัติที่ต้องการกำหนดให้กับ Selector
  • Value คือ ค่าที่กำหนดให้กับ Property หรือ Attribute

โครงสร้างของ CSS

การกำหนดค่าของ Property กับ Value จะแยกด้วยเครื่องหมาย : (Colon)
แต่ถ้ามีมากกว่า 1 Property ต้องแยกด้วยเครื่องหมาย ; (semi-colon)
ตัวอย่าง
selector {property : value ; property : value ; ...}
