CSS Variable


การสร้างตัวแปรใน css จะช่วยเพิ่มความสะดวกสบายในการกำหนด style ให้กับ element
กำหนดชื่อตัวแปรได้เอง ด้วยเครื่องหมาย -- ตามด้วยชื่อตัวแปร

วิธีใช้งาน
  1. สร้าง :root{} ขึ้นมา
  2. กำหนดชื่อและค่าตัวแปรใน root โดยชื่อตัวแปรจะมี -- นำหน้า
  3. เรียกใช้งานโดยใช้ :var(ชื่อตัวแปร);
  4. ตัวแปรสามารถอ้างอิงตัวแปรอื่นได้โดย var()
