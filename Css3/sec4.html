Display

Display คือการจัดรูปแบบการแสดงผล
  • None : ไม่มีการแสดงผล
  • Black : แสดงผลแบบ block ขึ้นบรรทัดใหม่ก่อน (เรียงในแนวตั้ง)
  • Inline : แสดงผลแบบ inline ไม่ขึ้นบรรทัดใหม่ (เรียงในแนวนอน)
  • inline-block : แสดงผลแบบแนวนอนและขยายพื้นที่ด้านใน
Visibility

แสดงหรือซ่อนวัตถุในหน้าเว็บ Tag Visibility
  • hidden ซ่อนวัตถุ
  • visible แสดงวัตถุ
การจัดตำแหน่งด้วย Position
position ใช้ในการจัดตำแหน่งของวัตถุ
  • static : วัดวางแบบปกติ
  • relative : กำหนดตำแหน่งวัตถุโดยนับจากจุดที่วัตถุนั้นๆอยู่
  • absolute : จัดวางวัตถุ ตำแหน่งใดก็ได้แต่ต้องกำหนดจากขอบของ Elememt ที่อยู่
  • fixed : ใช้ในการจัดวัตถุให้อยู่ตำแหน่งเดิม
  • sticky : ให้วัตถุติดขอบเมื่อเลื่อนไปถึง
Tag z-index : จัดตำแหน่งหน้าหลัง
