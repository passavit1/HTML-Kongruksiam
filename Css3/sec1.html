ตัวอย่าง

class aa

รูปแบบการประกาศใช้

แบบ Internal
  • inline เป็นการแทรก Attribute style ลงใน Tag HTML
  • Embeded การกำหนดรูปแบบ Style sheet ภายในไฟล์เดียวกันกับไฟล์เว็บประกาศไว้ที่ Head
แบบ External
กำหนดไฟล์ Style sheets โดยมีนามสกุล .css แล้วค่อยเชื่อมเข้ามาใช้ภายในเว็บเพจ

Internal / Embeded

กำหนด Style ไว้ที่ Head ของเว็บเพจ เพื่อกำหนด Style ให้กับ element ในเว็บเพจนั้น

External
กำหนด style sheet แยดเป็นอีกไฟล์ใช้นามสกุล .css แล้วเชื่อมโดยใช้ Tag < Link> เขียนในส่วนของ head แล้วกำหนดไฟล์ใน href

Units in CSS แบบตายตัว (absolute)

Units Description
px Pixel โดย 1PX = 0.75pt
pt Point โดย 1pt = 1/72 inchs
cm เซนติเมตร
mm มิลลิเมตร
in Inches (1in = 96px = 2.54cm)
pc picas (1PC = 12pt)

Units in css แบบอัตราส่วน (Relative)

units Description
% กำหนด % มักใช้กับความกว้างและความสูง
em element parent เป็นการกำหนดจำนวนเท่าขอขนาดปัจจุบัน
rem กำหนดขนาดอ้างอิงกับ root element ปกติค่า = 16 px
vw 1% หรือ 1/100 ของ viewpoint width
- width ของ browser viewpoint เท่ากับ 750px ค่า 1vm = 7.5px
vh 1% หรือ 1/100 ของ viewpoint height
- width ของ browser viewpoint เท่ากับ 900px ค่า 1vm = 9px
vmin , vmax กำหนดค่าต่ำสุดและสูงสุดของ viewppoint
