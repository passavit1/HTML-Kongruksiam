HTML DOM


เมื่อเว็บสร้างเรียบร้อย web browser จะสร้าง DOM ของหน้านั้นขึ้นมา
คุณสมบัติ HTML DOM
  • เข้าถึงและเปลี่ยนคุณสมบัติทั้งหมดของ Element
  • ควบคุมและเปลี่ยนรูปแบบ CSS ได้
  • สามารถตอบสนองกับเหตุการณืที่เกิดขึ้นหน้าเว็บได้
เข้าถึง element ผ่าน if , tag , class

  • document.getelementbyid('id name');
  • document.getelementbytagname('tag name');
  • document.getelementbyclassname('class name');

Dom document

  • เปลี่ยนเนื้อหา HTML : element.innerHTML
  • เปลี่ยนเนื้อหา TEXT : element.innertext
  • เปลี่ยน style element : element.style.properties = value

ดำเนินการผ่าน method
  • element.setattribute(attribute.value
