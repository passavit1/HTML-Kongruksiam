DOM Event

เหคุการณ์หรือการกระทำบางอย่าง ที่เกิดขึ้นกับ element
  • onfocus='' = เมื่อ element ได้รับการโฟกัส
  • onblur='' = เมื่อ element สูญเสียการโฟกัส
  • onchange='' = เมื่อมีการเปลี่ยนแปลงค่าในฟอร์มรับข้อมูล
  • onselect='' = เมื่อผู้ใช้เลือกข้อความ
  • onsubmit='' = เมื่อผู้ใช้คลิกปุ่ม submit
  • onlad='' = เมื่อทำการโหลดหน้าเว็บเพจ
  • onunload=''เมื่อยกเลิกการโหลด
  • onmouseover='' = เมื่อเมาส์อยู่บน object นั้นๆ
  • onmouseout='' = เมื่อเมาส์ถูกเลื่อนออก
  • onclick='' = เมื่อมีการกดคลิกเมาส์
