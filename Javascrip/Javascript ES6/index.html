ทำงานกับ array

เพิ่มและลบสมาชิกใน array
  • push = เพิ่มสมาชิกต่อท้าย
  • pop = ลบสมาชิกตัวหลังสุดออก
  • shift = ลบสมาชิดตัวด้านหน้าสุด
  • unshift = เพิ่มสมาชิกเข้าไปหน้าสุด
  • splice(ตำแหน่ง index ที่จะลบ,จำนวนที่จะลบ,สามารถแทรกสมาชิกในตำแหน่งที่ลบ)
  • slice(ตำแหน่งเริ่มต้น,ตำแหน่ง index สุดท้ายที่ต้องการใช้ +1) = ดึงช่วงสมาชิกออกมาใช้
การ loop ข้อมูลใน array
  • for loop (ใช้ break , continue ได้)
  • for each (ใช้ break , continue ไม่ได้)
  • for of (ใช้ break , continue ได้)
ค้นหาข้อมูลใน array
  • indexof = หาตำแหน่ง index ที่ต้องการค้น
  • find = ค้นหาว่ามี ข้อมูลใน array ไหม
  • findindex = ค้นหาตำแหน่งข้อมูลที่ค้นหาใน array
